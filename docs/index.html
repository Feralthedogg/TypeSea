Project target

Runtime typeguards shaped like a compiler pipeline.

TypeSea treats validation schemas as frontend syntax. Guards execute through an interpreter, runtime compiler, or AOT module emitter, and can also expose an optimized Sea-of-Nodes graph for introspection.

  • No runtime, peer, optional, or bundled dependencies in the published package.
  • No public unchecked boundary: external values enter as unknown.
  • Expected failures return immutable Result values and frozen diagnostics.
  • Hot is() validation avoids diagnostic allocation.

Quick start

Define a guard once, then choose the execution path.

The same guard can narrow through is(), return diagnostics through check(), emit JSON Schema, or compile into a generated validator. 

import { compile, t, toJsonSchema, type Infer } from "typesea";

const User = t.strictObject({
  id: t.string.uuid(),
  age: t.number.int().gte(0),
  role: t.union(t.literal("admin"), t.literal("user"))
});

type User = Infer<typeof User>;

const checked = User.check(input);
const FastUser = compile(User, { name: "isUser" });
const schema = toJsonSchema(User);

Architecture

Schema syntax lowers into a validation graph.

The public schema tree is kept for diagnostics and introspection. Execution uses a graph with shared predicate and value nodes so common checks can be reused.

Builder Immutable public guards validate construction inputs.
Schema Frozen tree stores semantics and diagnostic structure.
IR Sea-of-Nodes graph exposes loads, predicates, and keysets for inspection.
Optimize Reachability and dense node ids keep graph invariants simple.
Execute Interpreter, runtime compile, or standalone AOT module.

Ecosystem adapters

Structural adapters without importing frameworks.

Adapter functions return objects or functions shaped for the target ecosystem. TypeSea does not import tRPC, Fastify, or React Hook Form at runtime.

Adapter Export Behavior
tRPC toTrpcParser Returns a parser object that emits decoded values or TypeSeaAssertionError.
Fastify toFastifyRouteSchema Exports JSON Schema fragments for route body, querystring, params, or headers.
Fastify toFastifyValidatorCompiler Returns validators that produce { value } or { error }.
React Hook Form toReactHookFormResolver Returns an async resolver with TypeSea messages mapped to field errors.

Benchmarks

Benchmark scope is checked by the release gate.

TypeSea tracks two performance questions: interpreted versus compiled TypeSea, and a JSON-compatible comparison against Zod, Valibot, and Ajv. The comparison packages are development-only measurement tools. 

npm run bench -- --run

Release gate

Publishing requires the full local gate.

The release script runs source policy, typecheck, lint, tests, build, dist policy, public API snapshot, package contents, clean consumer install, benchmarks, and pack dry run. 

npm run release:check
